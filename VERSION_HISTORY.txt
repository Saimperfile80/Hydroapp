RELEASED FEATURES:

Scientific Modules:
✅ Theis Analysis (transient pump test, well function solution)
✅ Cooper-Jacob Analysis (semi-log approximation)
✅ Lefranc Test (charge/discharge in boreholes)
✅ Lugeon Test (rock injectability with pressure stepping)
✅ Porchet Test (loose formation permeability)
✅ Piezometric Analysis (water level time series)

AI Pedagogical Modules:
✅ Anomaly Detection (Z-score, IQR, spatial methods)
✅ Parameter Recommender (6 lithology types with K ranges)
✅ Pre-Calculation Validator (OK/ATTENTION/BLOQUÉ status)

User Interface:
✅ PySide6 main application window
