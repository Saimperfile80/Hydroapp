PROJECT OVERVIEW
═════════════════════════════════════════════════════════════════════════════

HydroAI is a pedagogical platform for hydrogeological modeling combining:
- Scientific rigor (6 calculation methods)
- Pedagogical AI guidance (not calculation replacement)
- Modern user interface (PySide6)
- Comprehensive documentation

Status: ✅ MVP READY FOR CLASSROOM USE
